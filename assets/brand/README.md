# Brand Assets

Replace these placeholder SVG files with your own brand logos to white-label the platform.

| File                  | Purpose                                  | Recommended Size         |
| --------------------- | ---------------------------------------- | ------------------------ |
| `brand-logo.svg`      | Main brand logo for light theme (header) | 200×48px, transparent bg |
| `brand-logo-dark.svg` | Main brand logo for dark theme (header)  | 200×48px, transparent bg |
| `platform-logo.svg`   | Platform icon (square)                   | 64×64px, transparent bg  |

These files are copied to `dist/brand/` at build time and served as static assets.

See [WHITE_LABEL.md](../../docs/WHITE_LABEL.md) for the full white-labeling guide.
